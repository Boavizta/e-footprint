Decisions log

  • Keep zstandard. Benchmark (2026-06-30): on near-incompressible float32 timeseries, zstd matches zlib's ratio (1.13×) but is ~10–14× faster; a batch of 100 five-year series is 27 ms (zstd) vs ~415 ms (zlib) vs 3.8 s (lzma) to decompress+recompress; importing zstandard costs ≈0 MB. Replacing it would regress the interface's hot path for no real gain.
  • Rolling window, no upper cap. Hard caps on a library propagate to consumers.
  • PEP 621 migration included. Verified (2026-06-30) that [project] + explicit classifiers suppresses poetry-core's auto-generation — otherwise (legacy layout / omitted) a release would advertise the untested :: 3.14. Modest, mostly-mechanical change (12 runtime deps; dev group untouched); the one risk — a caret→PEP 508 mistranslation changing resolution — is caught by a lockfile/export diff. Curated classifiers then reflect the tested set, replacing the earlier "add trove classifiers" item.
  • Parked (memory-optimization track, roadmap.md): viz (matplotlib/plotly/pyvis) and pandas as optional extras — measure the RAM win first; pandas needs a constitution §4 note.
