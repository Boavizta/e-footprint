    EdgeStorage

    Params

    name

    A human readable description of the object.

    storage_capacity_per_unit

    Storage capacity per unit of edge ssd storage in gigabyte_stored.

    carbon_footprint_fabrication_per_storage_capacity

    Fabrication carbon footprint per unit of edge ssd storage per storage capacity in kilogram / terabyte_stored.

    base_storage_need

    Edge ssd storage initial storage need in gigabyte_stored.

    lifespan

    Lifespan of edge ssd storage in year.

    nb_of_units

    Number of units of edge ssd storage in dimensionless.
