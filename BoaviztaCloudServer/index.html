    BoaviztaCloudServer

    Params

    name

    A human readable description of the object.

    provider

    Serverless cloud functions cloud provider.

    For example, scaleway.

    instance_type

    Serverless cloud functions instance type.

    For example, ent1-s.

    server_type

    Server type of serverless cloud functions.

    For example, serverless.

    lifespan

    Lifespan of serverless cloud functions in year.

    idle_power

    Idle power of serverless cloud functions in watt.

    power_usage_effectiveness

    Pue of serverless cloud functions in dimensionless.

    average_carbon_intensity

    Average carbon intensity of serverless cloud functions electricity in gram / kilowatt_hour.

    utilization_rate

    Serverless cloud functions utilization rate in dimensionless.

    base_ram_consumption

    Base ram consumption of serverless cloud functions in gigabyte_ram.

    base_compute_consumption

    Base cpu core consumption of serverless cloud functions in cpu_core.

    storage

    An instance of Storage.

    fixed_nb_of_instances

    User defined number of serverless cloud functions instances. Can be an EmptyExplainableObject in which case the optimum number of instances will be computed, or an ExplainableQuantity with a dimensionless value, in which case e-footprint will raise an error if the object needs more instances than available.
